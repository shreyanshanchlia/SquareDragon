Jumping Dinosaur

Basic integration of Unity Analytics which collects anonymous usage statistics. This data is used to know the number of daily active users, how many people reached a certain high score, which game mode is preferred and other similar information. This data is anonymous and is only used to help improve games. The same tools are also used to get performance reports so we can track and resolve game crashes and errors. I do not own the game idea. It is from the popular browser Chrome. No personal information is collected. Advertising This game incorporates rewarded video ads, interstitial ads, banner ads provided by unity as well as ad mob as a way to monetize. In order for advertising networks to display relevant ads to users, they use Advertising IDs. To turn off personalized ads please use your Android devices opt-out settings.
